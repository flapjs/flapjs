ExportData

An object map that represents the processed export data. These are used as arguments for a downloader to further evaluate and then finally download to the client's machine. Any additional properties are also passed to the downloader.
Type:
  • Object
Properties:
Name Type Description
name String the file name for the exported object
type String the download type (this is NOT the same as export type or file type)
data * the processed target data to put into a file
