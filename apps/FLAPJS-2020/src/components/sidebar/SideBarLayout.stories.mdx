# SideBarLayout

A React component that adds a sidebar with a viewport. The sidebar can be positioned
on any side of the viewport. The viewport rendered would be its nested children. If
you want to add an element to the sidebar itself, use the `sideBar` prop.

<Props of={SideBarLayout}/>

Here is an example string sidebar content.

<Preview>
    <Story name="with string" height="10rem">

                    This is some other content, rendered next to it.

But what if you want to have scrollable content in the viewport? By default,
it just hides overflow (or `hidden`). If you want to scroll, you should set
`viewOverflow` to `auto`.

<Preview>
    <Story name="viewport overflow" height="10rem">
